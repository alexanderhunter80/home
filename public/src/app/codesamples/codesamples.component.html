Recently, I spent some time flexing my Java skills by implementing a variety of data structures and comparing their performance on some basic operations.

Together with Chris Luttio, I built naive implementations of multiple such structures in Java. Then, from a hopefully-even playing field, we ran these structures through several test cases, gathering data on their time and memory performance through a small benchmarking service that I wrote for this project.

The tabs included here are the structures that I personally implemented - I have omitted those where Chris did the majority of the work. Where I used or adapted code from other sources, I have included references to the original source in the comments.

If you are interested in the results of our testing, that is available in a Github repository, both as raw data and as a LibreOffice Impress presentation.
